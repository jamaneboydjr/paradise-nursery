# Paradise Nursery Shopping Cart

A dynamic React shopping cart application for an online plant shop called **Paradise Nursery**.

## Features
- Landing page with background image and “Get Started” button
- Product listing page with plants grouped into categories
- Add to Cart + disabled button once added
- Navbar links: Home / Plants / Cart
- Cart icon shows dynamic total item count
- Cart page allows:
  - Increase/decrease item quantities
  - Remove items
  - Shows total cost per item and total cart amount
  - Checkout shows “Coming Soon”
  - Continue Shopping returns to product listing

## Tech Stack
- React
- React Router
- Redux Toolkit
- Vite (recommended)
